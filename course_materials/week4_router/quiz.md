# Week 4 Quiz: Router Networks

## Question 1: Routing Logic
**In a Router-Expert architecture, what is the role of the Router?**
- [ ] To process the final answer
- [ ] To classify the input query and assign it to the most suitable expert
- [ ] To translate the text
- [ ] To compress the data

<details>
<summary>Answer</summary>
**To classify the input query**. The router acts as a dispatcher.
</details>

## Question 2: BERT Embeddings
**Why do we use the [CLS] token embedding for classification in BERT?**
- [ ] It represents the aggregate representation of the entire sequence
- [ ] It is the first token
- [ ] It is the only token that attends to everything
- [ ] All of the above

<details>
<summary>Answer</summary>
**All of the above**. The [CLS] token is designed to capture sentence-level meaning.
</details>
