# Week 11 Quiz: Optimization

## Question 1: Quantization
**What is the primary benefit of INT8 quantization over FP32?**
- [ ] Higher accuracy
- [ ] 4x reduction in memory usage and faster matrix multiplication
- [ ] Better gradients
- [ ] Longer context window

<details>
<summary>Answer</summary>
**4x reduction in memory**. 8 bits vs 32 bits.
</details>
